ДОБАВЛЕН РЕЖИМ АВТОСИПОВ ПО МАССЕ
================================

Изменения:

1. cip_thresholds.json
   - Добавлено поле "mode": "штуки" | "масса"
   - Значение по умолчанию: "штуки"

2. product_density.json (НОВЫЙ ФАЙЛ)
   - Справочник плотности продуктов в формате:
     {
       "Тип продукта": плотность_кг_на_литр
     }
   - Используется для расчета массы при mode = "масса"

3. schedule_tab.py
   
   НОВЫЕ ФУНКЦИИ:
   - _load_product_density() - загрузка справочника плотности
   - _calculate_mass(job, density_map) - расчет массы продукта
     Формула: масса_кг = количество × объем_литров × плотность
   
   НОВЫЙ КЛАСС:
   - DensitySettingsWindow - окно настройки плотности продуктов
     * Позволяет редактировать плотность для существующих типов
     * Позволяет добавлять новые типы продуктов
     * Позволяет удалять типы
   
   ИЗМЕНЕНИЯ В ЛОГИКЕ:
   - _build_schedule_for_line():
     * Добавлено определение auto_cip_mode ("штуки" или "масса")
     * Загрузка справочника плотности при mode = "масса"
     * Расчет job_qty с учетом режима (штуки или кг)
     * Пересчет current_part_qty_pieces для корректного отображения
     * Передача unit в _create_auto_cip_record
   
   - _create_auto_cip_record():
     * Добавлен параметр unit для правильного отображения единиц
     * Название и примечание теперь включают unit (шт/кг)
   
   UI ИЗМЕНЕНИЯ:
   - AutoCipSettingsWindow:
     * Добавлен Combobox выбора режима ("штуки"/"масса")
     * Динамическая метка единиц измерения (штуки/кг)
     * Кнопка "Плотность продуктов" для открытия DensitySettingsWindow
     * Сохранение поля "mode" в настройках

ИСПОЛЬЗОВАНИЕ:

1. Открыть "Расписание" -> "⚙️ Авто-CIP"
2. Для каждой линии выбрать режим:
   - "штуки" - подсчет по штукам
   - "масса" - подсчет по массе (кг)
3. При режиме "масса" нажать "⚖️ Плотность продуктов"
4. Настроить плотность для типов продуктов
5. Сохранить настройки

ПРИМЕРЫ:

Режим "штуки":
- Порог: 30000 шт
- Задание: 6000 шт "Сироп 1.0л"
- Результат: без разбиения, накопление 6000 шт

Режим "масса":
- Порог: 3000 кг
- Задание: 6000 шт × 1.0 л × 1.3 кг/л = 7800 кг
- Результат: разбиение на 3 части, авто-CIP после каждой части
